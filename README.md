# CMBR
CMBR - Chess Move Binary Representation (WIP)

### Building
For release builds

```bash
cargo build --release
```

For development builds
```bash
cargo build
```

### Usage
For usage run
```bash
cmbrcc help
```

or

```bash
cargo run -- help
```

### Documentation
The documentation (WIP/Unfinished) is provided as a latex and [pdf](./docs/CMBR.pdf) file
