# CMBR
CMBR - Chess Move Binary Representation (WIP)

### Building
For release builds

```bash
cargo build --release
```

For development builds
```bash
cargo build
```

### Usage
For usage run
```bash
cmbrcc help
```

or

```bash
cargo run -- help
```
